# Projects

You may be involved in several projects,
each of which have their own repositories.
Use this directory to keep notes/plans,
collect links to papers and other resources for your personal use
(if they're relevant for the broader project, you can add them to notes in their repo instead),
or just link out to those other repos.

For more advanced users,
you can use git submodules to manage all of your projects
in the same directory.

## Usage

### Basic

To fully manage projects in other repositories,
simply replace this section with markdown links to your project(s), eg

- [My first project](https://github.com/bonhamlab/some-project)
- [Lab website project](https://github.com/bonhamlab/bonhamlab.github.io)
